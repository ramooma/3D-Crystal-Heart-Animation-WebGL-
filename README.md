# 💖 3D Crystal Heart Animation (WebGL)

### A visually stunning, math-driven 3D particle animation built with custom GLSL shaders.

## 📌 Project Overview
This project is an advanced exploration into **WebGL** and **3D computer graphics**. It features a mesmerizing, animated "Crystal Heart" made of glowing particles. 

Instead of importing a pre-made 3D model, the heart is generated entirely through code using mathematical parametric equations within custom Vertex and Fragment shaders. It also includes an interactive audio feature to play background music, creating a complete and immersive audiovisual experience.

## 🚀 Key Features
- **Custom Shaders (GLSL):** Written custom `x-vertex` and `x-fragment` shaders to control the exact position, color, and glow of every single particle.
- **Mathematical Rendering:** Utilizes the parametric equation of a heart to calculate the `$x, y, z$` coordinates of the particles dynamically in the 3D space.
- **Audio Integration:** Features a built-in, beautifully styled music player toggle (`#play-music`) that syncs the visual aesthetic with audio.
- **High-Performance Graphics:** Leverages the GPU through WebGL (via the `<canvas>` element) to render complex particle animations smoothly at 60 FPS.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **3D Graphics & Rendering:** WebGL, JavaScript
- **Shader Language:** GLSL (OpenGL Shading Language)
- **Concepts:** Particle Systems, 3D Mathematics, Shader Programming.
